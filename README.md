# DevIntel Developer Intelligence Platform - Dev Analysis 2

This repository contains the environment setup for Dev Analysis 2.

## Structure
- `data/`: Original, processed, and feature-engineered datasets.
- `notebooks/`: Jupyter notebooks.
- `src/`: Source code for data pipelines, modeling, NLP, and API backend.
- `frontend/`: React single-page application.
- `mlflow/`: Experiment tracking databases and artifacts.
- `monitoring/`: Evident AI model monitoring reports.
- `tests/`: Automated tests.
